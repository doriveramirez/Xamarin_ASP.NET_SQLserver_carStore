Modelos de coche


Para acceder a esta venta deberemos estar logeado. Una vez logeado desde la vista principal apretaremos el boton donde pone CRUD.



Una vez aqui pulsaremos la segunda opcion de la parte superior de la pantalla que pone "Car´s models o modelos de coche".



Una vez en esta vista podremos elegir la marca a la que pertenece este modelo, escribir el nombre del modelo,la potencia del coche, el color y el número de puertas. Ademas con los botones que tenemos en el recuadro podremos hacer las funciones de crear, modificar y eliminar ademas de limpiar lo que hemos escrito previamente.
