Marcas de coche


Para acceder a esta venta deberemos estar logeado. Una vez logeado desde la vista principal apretaremos el boton donde pone CRUD.



Una vez en esta vista podremos escribir el nombre de la marca,el lugar de la sede y el nombre del fundador. Ademas con los botones que tenemos en el recuadro podremos hacer las funciones de crear, modificar y eliminar ademas de limpiar lo que hemos escrito previamente.
